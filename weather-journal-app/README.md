# Weather-Journal App Project

## Overview
This Weather Journal app is created with Node.js & Express.

## Instructions
In this app you enter the zip code of the city you want to know its temperature right now, and you enter what you are feeling. Then, the application get the temperature information from OpenWeatherMap.com API. These information are stored in the database of the application.

## Important
This Version doesn't include node-modules folder, so please install (Express, cors, body-parser) before you start.
